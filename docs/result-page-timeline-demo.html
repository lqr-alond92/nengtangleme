这版的信息框架

1. 懂你的 AI 先用一句话定调：不是没资产，而是资产不能有效扛风险。
2. 理性的我 不先讲公式，先重构真假资产，拆穿账面资产错觉。
3. 关键证据 只展示 3 个最能证明结论的指标，不平铺 7 个。
4. 行动的我 把焦虑转成 3 个本周动作，并用一句心理锚点收束。
5. 完整精算 公式、代入过程和门槛全部保留，但默认收起。

设计原则

  1. 首页不展示公式。
  2. 首页不展示 7 个指标。
  3. 先讲资产错觉，再讲指标。
  4. 行动方案要比指标更醒目。

下一步可讨论

如果这个信息框架成立，下一轮再讨论视觉表现：是否加轻微时间线动效、真假资产是否用图形化对比、完整精算是否用底部弹层。

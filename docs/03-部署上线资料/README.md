## 使用建议

- 当前 Demo 出问题：先看 CloudBase 和 DeepSeek 代理文档。
- 要给体验用户发链接：先看手机 PWA Alpha 文档。
- 要讨论备案、域名、服务器租赁：看 `launch` 目录下的资料。

## 和 PRD 的区别

部署上线资料不定义产品功能，它只定义运行方式、云服务配置、域名和环境。

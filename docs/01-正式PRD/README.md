## 使用原则

1. 开发以 Approved 状态的 PRD 为准。
2. 重大方向变化不要覆盖旧版本，应新增版本。
3. 当前正在讨论的 AI 画像和大模型输出方案，稳定后应整理为 `PRD-001 v2.0`。

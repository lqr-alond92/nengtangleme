# AI 分析 Prompt 去繁就简方案

本方案基于两类参考模板整合而来：

1. 当下家庭财务体检：看资产、负债、收支、流动性和风险。
2. 全生涯供需缺口测算：把未来目标按折现率换算到今天，判断长期资源是否够。

我们的产品不直接复刻长报告，而是压缩成一个用户能看懂、能行动、能继续追问的 AI 分析。

## 一句话定位

给用户的不是财务报表，而是一份“穿越时间后的家庭财务判断”：

把用户现在真正能动用的钱、未来可赚到并折回今天的钱，以及未来目标折回今天后的成本放在同一张表里，判断：

- 当前赚钱能力能否支撑目标；
- 长远规划有没有明显风险；
- 如果有风险，现在最该改什么。

## 设计原则

- 用户输入保持轻：只收集 MVP 家庭财务画像。
- 计算逻辑藏在幕后：系统计算基础指标和折现后的供需关系。
- 页面输出交给大模型：让 AI 基于画像生成千人千面的分析，包括底部继续追问问题。
- 输出必须短而有力：不做七大比率长表，不做四层长报告。
- 建议必须可执行：不说“少花钱、多赚钱、提高收益率”这种空话。
- 不使用用户不容易理解的“能不能躺”作为核心判断话术，改为直接说明目标、收入和长期风险。

### 3. 计算口径

- 灵活资产：现金、存款、基金、股票、理财等相对容易动用的钱。
- 固定资产：房产、车位、长期资产等不容易快速变现的钱。
- 年度结余：年收入 - 年支出。
- 未来收入现值：把预计工作年限内的未来收入按折现率折回今天。
- 未来支出现值：把预计工作年限内的未来支出按折现率折回今天。
- 未来结余现值：未来收入现值 - 未来支出现值。
- 重构后可用资产现值：灵活资产 + 未来结余现值。
- 未来目标现值：把未来目标金额按折现率折回今天。
- 必须目标和想要目标分开判断。
- 固定资产默认不计入重构后可用资产，除非用户明确愿意出售、抵押或出租。

## 核心分析命题

页面不再拆成“现在的底盘”和“穿越时间后的判断”。这两件事合并为一个命题：

**重构真假资产。**

真实可用于支撑目标的资产，不是用户账面上的总资产，而是：

```text
重构后可用资产现值 = 当前灵活资产 + 未来结余折现值
```

其中：

```text
未来结余折现值 = 未来收入折现值 - 未来支出折现值
```

再用它对比：

```text
未来目标现值 = 必须目标现值 + 想要目标现值
```

由此得到核心判断：

```text
缺口或余量 = 重构后可用资产现值 - 未来目标现值
```

这就是“穿越时间”的价值：不是只看今天有多少钱，而是把未来赚钱能力、未来生活支出和未来目标都折算回今天，判断这套长期规划是否成立。

## Prompt 核心指令草案

```text
你是一名家庭财务分析助手。你的任务不是输出复杂财务报表，而是把用户的家庭财务画像翻译成一份能看懂、能行动、能穿越时间的判断。

请基于输入的用户画像和系统计算结果，完成三件事：

1. 重构用户的真实可用资产：
   不要直接把固定资产算作可用资产。请优先使用“当前灵活资产 + 未来结余折现值”作为重构后的可用资产现值。

2. 判断当前赚钱能力能否支撑未来目标：
   重点看未来收入折现值、未来支出折现值、未来结余折现值、目标现值、必须目标和想要目标的缺口或余量。

3. 明确长期规划风险和后续 Action：
   风险必须来自用户数据，Action 必须包含明确后续步骤和具体行动点。禁止只说“少花钱”“多赚钱”“提高收益率”。

表达要求：
- 用普通人能听懂的话。
- 必须引用用户的关键数字。
- headline 必须一针见血地回答两件事：当前赚钱能力能否支撑目标；长期规划风险大不大。
- 不要使用“能不能躺”“不是不能躺”“马上能不能躺”这类用户不容易理解的话术。
- 必须区分“账面资产”和“重构后真实可用资产”。
- 必须说明未来目标是宽松、紧张还是有缺口，并说明原因。
- 底部 followUpQuestions 必须由模型根据用户实际情况生成，不要使用固定问题。
- 不要输出七大比率长表。
- 不要输出泛泛的理财常识。
- 不要承诺投资收益。
- 不要制造恐慌，结尾要让用户知道下一步怎么做。

输出必须是 JSON，字段包括：
headline, coreConclusion, reconstructedAssets, risks, actions, followUpQuestions。
```

## 页面展示建议

结果页不再堆指标卡，而是展示 5 个区块：

1. 一句话结论
2. 重构后的真实资产
3. 长期规划风险
4. 后续步骤
5. 具体行动点

底部保留继续追问，但追问问题必须由大模型根据用户实际画像动态生成。页面不写死固定问题。

追问问题应围绕用户当前最关键的不确定性，例如：

- 固定资产是否可动用；
- 未来收入是否稳定；
- 某个目标是否需要降级或延期；
- 必须目标和想要目标之间如何排序；
- 如果收入下降、支出上升或目标提前，缺口会如何变化。

## 和参考模板的取舍

保留：

- 真假资产/生息资产的思想。
- 流动性和负债压力判断。
- 折现率和目标现值法。
- 未来收入、未来支出、未来结余折现后的供需轧差。
- 大白话解释和行动建议。

删减：

- 七大财务比率完整长表。
- 大量公式过程展示。
- 冷血精算师式长篇报告。
- 四层报告结构。
- 需要用户手动填很多细项的输入方式。

## 第一版结论

第一版 AI 分析不做“全量财务体检报告”，而做“重构真假资产后的长期规划判断”。

用户只需要填 MVP 画像；系统把当前灵活资产、未来收入、未来支出和未来目标全部折算到今天；大模型负责把这些数据转成个性化判断、风险分析、行动建议和动态追问。

# 04 原型与工作流

这个文件夹用于汇总 HTML 原型、交互探索和团队工作流文档。它们用于看效果、讨论体验和管理任务，不是正式 PRD。

## HTML 原型

- [`../ai-judgement-chat-demo.html`](../ai-judgement-chat-demo.html)：AI 判断聊天式结果页原型。
- [`../onboarding-interaction-flow.html`](../onboarding-interaction-flow.html)：首次进入和输入流程原型。
- [`../onboarding-visual-proposal.html`](../onboarding-visual-proposal.html)：视觉方向原型。
- [`../original-interaction-clickable.html`](../original-interaction-clickable.html)：早期可点击交互原型。
- [`../result-page-timeline-demo.html`](../result-page-timeline-demo.html)：结果页时间线原型。
- [`intake-smart-composer-demo.html`](intake-smart-composer-demo.html)：输入引导重组原型，演示底部智能填写栏、条件展开和目标频率选择。
- [`result-page-ai-rework-demo.html`](result-page-ai-rework-demo.html)：AI 结果页重构原型，包含打字机动效、真实资产、核心结论、风险和行动卡片。

## 工作流与模板

- [`../workflow/ai-product-team-workflow.md`](../workflow/ai-product-team-workflow.md)：AI 产品团队工作流。
- [`../workflow/task-template.md`](../workflow/task-template.md)：任务模板。

## 使用建议

- 想看某个交互长什么样：看 HTML 原型。
- 想整理任务或测试：看工作流模板。
- 原型里的方案确认后，应沉淀到产品迭代方案或正式 PRD。

# 能躺了吗文档导航

这个目录下的资料按用途分成四类。为了避免旧链接失效，原始文件暂时不搬动；下面四个文件夹先作为导航索引使用。

## 1. 正式 PRD

入口：[`01-正式PRD/README.md`](01-正式PRD/README.md)

用途：记录已经确认或准备进入开发的产品需求。开发时优先看这里。

## 2. 产品迭代方案

入口：[`02-产品迭代方案/README.md`](02-产品迭代方案/README.md)

用途：记录正在讨论的新方向、下一版产品策略、AI 画像和 Prompt 方案。确认后再升级成正式 PRD。

## 3. 部署上线资料

入口：[`03-部署上线资料/README.md`](03-部署上线资料/README.md)

用途：记录手机 Demo、CloudBase、DeepSeek 代理、域名备案、正式上线等工程和运维资料。

## 4. 原型与工作流

入口：[`04-原型与工作流/README.md`](04-原型与工作流/README.md)

用途：记录 HTML 原型、交互探索、产品团队工作流和任务模板。

## 查看建议

- 要看产品当前依据：先看正式 PRD。
- 要看下一步要改什么：看产品迭代方案。
- 要部署或排查线上问题：看部署上线资料。
- 要回看原型或工作方法：看原型与工作流。
